# Graduação em Análise e Desenvolvimento de Sistemas (ADS) 🎓

Este repositório centraliza todas as atividades práticas, exercícios de lógica, projetos interdisciplinares e laboratórios desenvolvidos ao longo da minha graduação na **Estácio**.

O objetivo é documentar minha evolução técnica, manter um histórico organizado das disciplinas e servir como portfólio para conceitos fundamentais de computação.

---

## 📂 Estrutura de Organização

O repositório está organizado por semestres e, dentro de cada um, por disciplinas:

## 🛠️ Tecnologias e Ferramentas Estudadas

Ao longo do curso, utilizo diversas tecnologias dependendo da grade curricular:

* **Versionamento:** Git e GitHub.

## 🚀 Como navegar pelos códigos

Para explorar as atividades de uma matéria específica:
1. Navegue até a pasta do **Semestre** desejado.
2. Acesse a pasta da **Matéria**.
3. Lá você encontrará os arquivos fonte (ex: `.c`) e, em alguns casos, um README específico explicando o exercício proposto pelo professor.
